# コンセプト

## 仮タイトル

剥ぎ取りダンジョン

## ゲーム概要

地下100階まで続くダンジョンを探索し、敵を倒し、装備用素材を剥ぎ取りながら奥へ進むローグライク風ゲーム。地下100階にいるラスボスを倒すとクリア。

## 面白さの中心

- 敵から何を剥ぎ取るか選ぶ
- 剥ぎ取った素材で装備の作成や強化につなげる
- リスクを取って深く潜るか、帰還するかを判断する
- 地下100階のラスボス撃破を目指す

## 最初の試作で確認したいこと

- 移動して敵とぶつかるだけで最低限楽しいか
- 敵を倒したあとの剥ぎ取り選択が面白いか
- 装備素材を集める目的がわかりやすいか
- 階層を進む目標がわかりやすいか
- 1プレイを短く区切れるか
