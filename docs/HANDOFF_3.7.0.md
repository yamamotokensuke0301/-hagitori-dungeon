## 3.7.0で入った主な変更

- ダンジョンユニークは当代の冒険者につき一度だけ出現。1体討伐で調査度5/5と心を獲得し、死亡時はユニーク調査・心・討伐歴だけをリセット。
- 通常箱と宝物庫へ、適正階層より上位の魔法書が0.01%で出る先取り抽選を追加。
- 宿屋の助言45件を全面確認し、共通死亡コメントを8件から32件へ拡張。
- 488体のユニーク台詞を最終表示名へ同期。改名個体147体は気質・欲求・形見・秘密・技名も名前由来へ再構成。
- 固定アーティファクト名30件をカタカナ混じりへ変更。
- 固定アーティファクトを65種類へ拡張。`★やばすぎるビキニ上♡`と`★やばすぎるビキニ下♡`は防御0・各6属性免疫。どちらか一方でも装備すると、生成・増援・セーブ再開を含むダンジョン内の全モンスターが起きる。
- 固定アーティファクトは黒星`★`、ランダム一点物は白星`☆`、艶装備は末尾に`♡`を表示。

## 次のBGM作業

ユーザーがクラウドコード側で`abyss-bgm`と`tension-5-bgm`だけを練り直す。他の8曲、BGMの選曲ロジック、M4A参照は変更しない。

作曲方針:

- 通常の機能和声から意図的に外し、ベースと主旋律を安易に調和させない。
- 各パートは単純に保ち、異なる周期の重なりで固有性を作る。
- `tension-5`: ドラム4拍、ベース5拍、主旋律7拍を第一候補とする。最大危険度なので強い不一致を許容する。
- `abyss`: 同じ発想を控えめに使い、長時間探索で疲れないよう低音・鐘・合唱の位相を緩やかにずらす。
- 「もっと個性的に」という抽象指示ではなく、使用音集合、周期、アクセント、一致地点をコード上の規則として固定する。

現行音圧の参考値:

- `abyss-bgm.m4a`: 約-16.9 LUFS、True Peak約-1.1 dBTP、LRA約5.6 LU。
- `tension-5-bgm.m4a`: 約-15.0 LUFS、True Peak約-0.7 dBTP、LRA約5.1 LU。

`tools/generate_bgm.py`は生成元WAVを出力する。完成後は対象2曲だけをAAC 192kbpsへ変換し、次のファイルを上書きする。

```text
assets/audio/abyss-bgm.m4a
assets/audio/tension-5-bgm.m4a
```

生成途中のWAVはコミットしない。`js/main.js`の参照先はすでにM4Aなので変更不要。作業後は次を実行する。

```sh
osascript -l JavaScript tools/smoke_test.js
git diff --check
```
