# 正式版1.0.1 修正内容

2026-07-13、正式版1.0.0のリリース監査から引き継いだ不具合を修正した。

## セーブ

- セーブ初期化後もリビジョンを単調増加させ、開いたままの別タブが旧データを復活させないようにした
- 破損セーブから新規状態へ復旧する際、残存リビジョンを引き継いで以後も保存できるようにした
- 不正な敵レコードと、更新で廃止されたモンスターIDだけを除去し、進行中ダンジョン全体は維持するようにした
- 除去対象がフロア守護者だった場合は、進行不能を避けるため歩行可能な下り階段を復元する

## 戦闘

- 壁内にいる壁抜けモンスターへ、非幽霊の冒険者も隣接近接攻撃できるようにした
- 敵への鈍化・拘束・感電・被害増幅・防御侵食・攻撃減衰・幻惑を、加速度による行動回数ではなく世界ターン単位で減算するように統一した
- 花ペットへ移動した時は位置を交換し、一マス通路を塞ぎ続けないようにした

## ダンジョン生成

- 出発・階段下降では次フロアの生成成功後にだけ、階層・最深到達・回復・商店解禁・ダンジョン本体を更新するようにした
- フロア生成中に調査記録やログを変更しないようにし、生成失敗を完全に巻き戻せるようにした
- 狂気の部屋の敵数を空きマス数以下に制限し、配置関数にも位置不足の防御を追加した
- 壁内で守護者を倒した場合、そのマスを床へ変えてから下り階段を置くようにした
- 異変発生後に現れる増援・召喚・宝箱奇襲・罠召喚にも、現在の異変による能力補正を適用するようにした

`deferredSpecialRooms` は同時当選した特別室の配置断念を検査する診断記録として使用中のため、プレイ不具合ではないと判断して維持した。

## 検証

- 全スモークテスト合格
- 複数タブ初期化、破損セーブ復旧、未知モンスターID、壁内守護者、加速敵の状態異常、花ペット通路、異変後召喚、生成強制失敗の回帰テストを追加
