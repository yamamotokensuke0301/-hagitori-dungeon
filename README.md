# Hagitori Dungeon

Current version: **Prototype 3.3.0**

『剥ぎ取りダンジョン』は、地下100階対応を前提にしたスマートフォン向けターン制ローグライクRPGのブラウザプロトタイプです。

現在の実装は地下1階から100階まで対応しています。戦闘、回数制の剥ぎ取り、素材流通、宝箱探索、魔法習得、複合属性耐性、加速度、装備購入を組み合わせて攻略します。

## 起動方法

プロジェクト直下で静的サーバーを起動します。

```bash
python3 -m http.server 5174 -d .
```

ブラウザで開きます。

```text
http://127.0.0.1:5174/index.html
```

単体HTMLとしてではなく、通常のWebサーバー配下に置く構成を想定しています。

## GitHub Pagesで公開

`main` ブランチへのpushで、構文検査後に現行ルートだけをGitHub Pagesへ自動公開するワークフローを用意しています。GitHub側では `Settings` → `Pages` → `Source: GitHub Actions` を選択してください。

初回公開の詳しい手順と公開後チェックは [docs/GITHUB_PAGES.md](docs/GITHUB_PAGES.md) を参照してください。`src/` と `publish/` は旧版なので公開元に使用しません。

## 操作

- 街: 職業ごとに独立したレベル・経験値を持つ転職、宿屋、自宅、商店、調査所、ダンジョン入口を使う
- 宿屋: 10GでHPと一時的な能力低下を回復し、戦闘・罠・職業・報酬・深層探索を扱う45種類の助言から、直近6件と重複しない話を聞く
- 自宅: 冒険者情報と通常攻撃期待値、生い立ちを確認・編集し、装備の入れ替え、魔法書の読解、使用魔法の選択を行う
- ダンジョン: 主人公を中心に照らした13×13表示を、8方向ボタン、矢印キー、テンキー、ローグライク式キーで移動
- 敵にぶつかる: マップ上で自動的に通常攻撃し、生き残った敵は反撃する
- 遠隔攻撃: 遠隔職は射程と射線が通る離れた敵を直接タップして攻撃する
- 剥ぎ取り: 撃破後の遺体に隣接すると中央の「待」が「剥」に変わる。一般モンスターは1〜2回、ユニークは3〜5回剥ぎ取れるが、残り回数は表示されない。盗賊だけは1行動で残りをすべて剥ぎ取る
- 宝箱: 魔法書かガラクタが入り、通常箱1%・宝物庫5%で★固定アーティファクト、さらに激レアで☆ランダム生成アーティファクト、まれにユニークモンスターが現れる。モンスター素材は入らない
- 職業技: ダンジョンでは「技」を構えて光る敵を選ぶ。闘技場では戦場ヘッダーの「技」から発動し、素材条件になる切断撃・精密射撃なども使える
- 戦闘: 踏み込み通常攻撃、遠隔攻撃、職業固有技、習得魔法を使い分け、闘技場では防御も使う。とどめを刺した攻撃も1行動として世界時間が進む
- 戦闘ログ: 下部の「全履歴」から、保存中の最大60件を新しい順に確認する
- 探索・罠解除: 遺体がない時の中央ボタンで周囲を調べる。罠は踏むと発見済みの作動可能状態で残り、隣接して解除できる。成功率は危険度、器用さ、運、職業、性格、種族で変わり、盗賊と便利屋が得意。解除失敗時は半分の効果で作動し、次回成功率が15ポイント上がる。3回作動すると壊れる
- 階段: 通常階には4〜6か所。十階層ごとの守護者を倒すと撃破地点に現れ、現在の冒険者は街からその守護者階へ直接出発できる。死亡すると深度標は失われる
- 帰還: 素材と宝箱の品を持って街へ戻る。回収した賞金首の報酬はギルドで受け取る
- 音楽: 初期ON。最初のタップまたはキー入力で鳴り始める
- 右上の音楽ボタン: 初回はBGM再生、再生中はBGMと効果音のON/OFF

## 実装済み

- 各曲専用編曲、4部展開、ステレオ残響とマスタリングで再構成した街・浅層・深層・最深層・修練連武闘技場・ボスの6曲
- 各階の通常生成上限と生存中の敵戦力を比較し、既存の階層曲と専用4曲で平静・不穏・緊迫・死線・破局を切り替える5段階動的緊張BGM
- 15属性、罠、転移、透明発見、摂食、売買、異変などを聞き分けられる57種類の個別効果音
- 街とダンジョンの行き来
- 14職業（攻撃した魔物を花ペットへ洗脳する「お花使い」、初期加速度+30の最強格「忍者」、摂食成長する「スカベンジャー」、万能型「便利屋」、回復魔法を使う「プリースト」を含む）
- 30種族（エルフの完全上位「ハイエルフ」、人間の完全上位「超人」、最弱の「スライム」を含む）と、ひらがな名13種類の性格による初期能力・レベル成長補正。「すけべ」は♡印の艶装備で共鳴する
- 主人公の名前入力と「ゆうかんなエルフたかし」形式の表示
- 名前・性格・種族から生成され、自宅で自由に編集できる冒険者の生い立ち
- アーチャー・魔法職の射線と射程を使った、敵マス直接タップによる遠隔攻撃
- 高レベル超能力者の時間停止と、深層ユニークの時間停止攻撃
- 地下1階から100階までの36×36〜60×60可変ランダム生成ダンジョンと、円形に広がる光源、約25%の常時照明部屋。常時照明は各入口から通路の最初の1マスまで届く
- 連結遺構、蟻道迷宮、巨間回廊、放射聖堂、環状遺跡、遠路坑道の6種類の生成構造。通路は原則1マス幅で、9室以上の構成へ極狭房と7〜11マス級の大部屋を適度に混ぜる
- 面積に応じた初期敵配置、浅層3〜6.5%・深層4〜8%の密度帯、敵のいない開始室、極狭房1体・通常室10マス当たり1体の上限
- 守護者を撃破したB10F、B20F……から現在の冒険者が選んで再挑戦できる、死亡時リセットのチェックポイント
- 10階ごとのランダムなフロア守護者と、撃破地点に出現する下り階段
- ダンジョン投影の内部左端に枠なしで浮かぶ縦型能力値表示。探索中はゴールド、外部能力値、装備HUDを表示しない
- 793種類のモンスター（一般305体、ユニーク488体、ダンジョン固有ユニーク388体、修練連武闘技場限定100体、金を盗んで転移逃走する希少種8体、B1F雑魚12種）
- 上位種エルフ族7体と、最強格のドラゴン族・悪魔族・天使族各7体。各種族に白から虹まで全色階級が存在する
- 全モンスターを種族へ分類し、同じ基礎種族は同じ一文字で表示。白・青・緑・黄・赤・紫・虹の7色階級で種族内の強さを示し、虹階級は文字色が刻々と変化する
- 16種族ごとの通常・レア・超レア・ウルトラレア剥ぎ取り素材。弱点・精密撃破、ユニーク、上位色、運で最上位素材の抽選率が上がる

- 9×9の移動・射程・射線戦を画面内固定操作で行う100連戦の修練連武闘技場。旧192体を10の強さ帯で監査し、属性・称号・個体核の重複が薄い100体へ選抜。種族共通文字と色階級で系統と強さを識別し、自己最高更新戦だけG・XP満額とGPを得て、再戦はG・XP半額・GPなし

## 開発メモ

街の「管理」から開発者パネルを開くと、レベル・所持金・GP・能力補正・出発階層を変更できるほか、装備や宝箱アイテムの付与、深度と調査記録の全解禁を行えます。変更内容は現在のセーブデータへ保存されるため、検証用セーブでの使用を推奨します。

- 基準仕様は `docs/GAME_DESIGN.md`
- データ形式は `docs/DATA_SCHEMA.md`
- 実行対象とモジュール責務は `docs/CODE_STRUCTURE.md`
- 未実装、簡略化、将来拡張は `docs/TODO.md`
- BGMの生成元は `tools/generate_bgm.py`
- 緊張度BGMだけを再生成する場合は `python3 tools/generate_bgm.py --tension-only`
- macOSでは `osascript -l JavaScript tools/smoke_test.js` で構文、ロスター件数、全30,160台詞の衝突、57効果音、剥ぎ取り回数、職業技、生成密度・特別室・罠安全域、魔法書、固定／ランダムアーティファクト、賞金・商店・闘技場報酬、死亡継承境界、旧セーブ移行を確認できる
