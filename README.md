# Hagitori Dungeon

『剥ぎ取りダンジョン』は、地下100階対応を前提にしたスマートフォン向けターン制ローグライクRPGのブラウザプロトタイプです。

現在の実装は地下1階から100階まで対応しています。戦闘、回数制の剥ぎ取り、素材流通、宝箱探索、魔法習得、複合属性耐性、加速度、装備購入を組み合わせて攻略します。

## 起動方法

プロジェクト直下で静的サーバーを起動します。

```bash
python3 -m http.server 5174 -d .
```

ブラウザで開きます。

```text
http://127.0.0.1:5174/index.html
```

単体HTMLとしてではなく、通常のWebサーバー配下に置く構成を想定しています。

## GitHub Pagesで公開

`main` ブランチへのpushで、構文検査後に現行ルートだけをGitHub Pagesへ自動公開するワークフローを用意しています。GitHub側では `Settings` → `Pages` → `Source: GitHub Actions` を選択してください。

初回公開の詳しい手順と公開後チェックは [docs/GITHUB_PAGES.md](docs/GITHUB_PAGES.md) を参照してください。`src/` と `publish/` は旧版なので公開元に使用しません。

## 操作

- 街: 職業選択、宿屋、自宅、商店、調査所、ダンジョン入口を使う
- 宿屋: 街へ帰還した後、休むとHPが完全回復する
- 自宅: 冒険者情報と通常攻撃期待値を確認し、装備の入れ替え、魔法書の読解、使用魔法の選択を行う
- ダンジョン: 主人公を中心に照らした13×13表示を、8方向ボタン、矢印キー、テンキー、ローグライク式キーで移動
- 敵にぶつかる: マップ上で自動的に通常攻撃し、生き残った敵は反撃する
- 遠隔攻撃: 遠隔職は射程と射線が通る離れた敵を直接タップして攻撃する
- 剥ぎ取り: 撃破後の遺体に隣接すると中央の「待」が「剥」に変わる。一般モンスターは1〜2回、ユニークは3〜5回剥ぎ取れるが、残り回数は表示されない
- 宝箱: 魔法書かガラクタが入り、ごくまれに★アーティファクトやユニークモンスターが現れる。モンスター素材は入らない
- 職業技: ダンジョンでは「技」を構えて光る敵を選ぶ。闘技場では戦場ヘッダーの「技」から発動し、素材条件になる切断撃・精密射撃なども使える
- 戦闘: 踏み込み通常攻撃、遠隔攻撃、職業固有技、習得魔法を使い分け、闘技場では防御も使う
- 階段: 次の階へ進む
- 帰還: 素材と宝箱の品を持って街へ戻る。回収した賞金首の報酬はギルドで受け取る
- 音楽: 初期ON。最初のタップまたはキー入力で鳴り始める
- 右上の音楽ボタン: 初回はBGM再生、再生中はBGMと効果音のON/OFF

## 実装済み

- 各曲専用編曲、4部展開、ステレオ残響とマスタリングで再構成した街・浅層・深層・最深層・修練連武闘技場・ボスの6曲
- 各階の通常生成上限と生存中の敵戦力を比較し、既存の階層曲と専用4曲で平静・不穏・緊迫・死線・破局を切り替える5段階動的緊張BGM
- 15属性、罠、転移、透明発見、摂食、売買、異変などを聞き分けられる57種類の個別効果音
- 街とダンジョンの行き来
- 12職業（摂食成長する「スカベンジャー」、万能型「便利屋」、回復魔法を使う「プリースト」を含む）
- 27種族と、ひらがな名10種類の性格による初期能力・レベル成長補正
- 主人公の名前入力と「ゆうかんなエルフたかし」形式の表示
- アーチャー・魔法職の射線と射程を使った、敵マス直接タップによる遠隔攻撃
- 高レベル超能力者の時間停止と、深層ユニークの時間停止攻撃
- 地下1階から100階までの48×48ランダム生成ダンジョンと、主人公の光源で中心から照らす13×13探索表示
- 生存中に到達した10階単位の深度標から再挑戦できるチェックポイント
- 10階ごとのランダムなフロア守護者と、撃破地点に出現する下り階段
- 部屋と通路型マップ
- 949種類のモンスター（ユニーク580体、ダンジョン固有388体、修練連武闘技場限定192体、B1F雑魚12種）
- 闘技場からダンジョンへ移籍し、階層用に再調整された150体と、20生態系×10個体核からなる新規ダンジョンユニーク200体
- 9×9の移動・射程・射線戦を画面内固定操作で行う192連戦の修練連武闘技場と、重複しない1文字かな・文字色による個体識別
- 地下5階の賞金首「赤熱のガルム」
- 1対1ターン制戦闘
- 光、闇、火、水、土、風、雷、毒、凍、呪、酸、斬、打、幻、鋼の15属性
- 一つの装備が複数の攻撃属性を持ち、対象へ最も有効な属性でダメージを与える複合攻撃属性
- 攻撃属性と職業固有技による素材変化
- 撃破後に残る遺体と、一般1〜2回・ユニーク3〜5回を内部抽選する、残数非表示の剥ぎ取りアクション
- 剥ぎ取り量が多い個体と、経験値が多い個体
- 降り階段の専用グラフィック、下降効果音、死亡時の暗転演出
- タップで閉じられる、短く落ち着いたレベルアップ演出
- 素材売却による商店装備の流通解禁
- 調査所で情報を購入する賞金首台帳、遺体の持ち帰り、ギルドの報酬受取窓口
- ギルドで確認できる賞金首の推定出現階層
- 職業ごとの武器・防具適性を持つ1,200種類超の装備と自宅の装備保管棚
- 装備引取で得るギルドポイントと、必要GPを明示した40種類の景品
- 宝箱限定の固定アーティファクト20種類。名称先頭の★表示、ネタ・ゴミ・平凡・使える・チートレベルの5分類、呪い、呪耐性によるペナルティ軽減、ギルド限定のGP納品
- 初級・中級・上級・最上級・禁断の5ランクを持つ魔法書。対応する魔法職が自宅で読むと1冊消費して魔法を習得し、読まずに商店で売却可能
- 15種類の売却用ガラクタと、魔法書・ガラクタ・アーティファクト・ユニーク奇襲だけを出す宝箱
- 複合属性耐性、希少な属性免疫、加速度、レア再生装備
- 火耐性装備による地下5階攻略導線
- 30件単位のページ送り、名前・ID検索、並び替え、表示絞り込み付きモンスター調査記録と、独立した死亡歴
- 自宅で確認できる通常攻撃期待値、威力幅、命中・会心、期待ヒット数、攻撃属性・射程、防御・回避、運搬情報
- 目撃から完全解析まで進む5段階のモンスター調査度（5/5でMAX）

- 素材重量による加速度低下、各階約21%で配置される罠、盗賊による隣接罠発見
- 遠隔・経験値吸収・吹き飛ばし・自爆・能力低下・所持品捕食を行う深層モンスター
- 全属性表示の色分けと、深層危険個体の虹色文字
- 遭遇、追跡、攻撃、被弾、回避、瀕死、奥義、特殊行動、調査、敗北など13文脈で話す全580体のユニークモンスター

- 王侯・獣・託宣者・道化・武人・怪異・狂信者・学者・守護者・賭博師・詩人・職人に分かれる気質、名称、台詞、色演出

- 宝箱からごくまれに飛び出す、その階層に対応したユニークモンスター

- セーブ初期化確認

## 開発メモ

- 基準仕様は `docs/GAME_DESIGN.md`
- データ形式は `docs/DATA_SCHEMA.md`
- 実行対象とモジュール責務は `docs/CODE_STRUCTURE.md`
